# CS50W Mail App

## 📌 Description
The CS50W Mail App is a full-stack web application that simulates an email client. It allows users to send, receive, archive, and manage emails through a dynamic interface. This project was built as part of the CS50 Web Programming course.

## 🚀 Features
- Send and receive emails between users
- Inbox, Sent, and Archived mailboxes
- Archive and unarchive functionality
- Mark emails as read and unread
- Dynamic email loading without page refresh

## 🛠️ Technologies Used
- Python (Django)
- JavaScript (DOM Manipulation, Fetch API)
- HTML5
- CSS3
- SQLite (Django default database)

## ▶️ How to Run Locally
1. Install Django:
   pip install django

2. Apply migrations:
   python manage.py migrate

3. Run the development server:
   python manage.py runserver

4. Open in browser:
   http://127.0.0.1:8000/
